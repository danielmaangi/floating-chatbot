## Quick Start

1. Clone the repository
2. Create a `.env` file based on `.env.example`
3. Add your OpenAI API key to the `.env` file
4. Install dependencies:
   ```bash
   npm install
   ```
5. Start the development server:
   ```bash
   npm run dev
   ```
